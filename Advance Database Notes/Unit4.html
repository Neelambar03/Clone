Unit IV: NoSQL Database Management

1. Introduction

NoSQL means “Not Only SQL.” It refers to databases that manage data without traditional relational tables. They are designed for flexibility, scalability, and high performance with large or unstructured data.

2. Need for NoSQL

  • Handles Big Data efficiently.
  • Schema-free and flexible.
  • Supports horizontal scaling across servers.
  • Ideal for real-time web and cloud apps.
  • Ensures high speed and availability.

3. Types of NoSQL Databases

4. MongoDB Basics

MongoDB is a document-oriented NoSQL database using BSON (Binary JSON). It is schema-less, scalable, and easy to use.

5. Common MongoDB Data Types

String, Integer, Double, Boolean, Array, Object, ObjectId, Date, Null, Binary.

6. CRUD Operations

OperationExample Command
Createdb.students.insertOne({...})
Readdb.students.find({ age: { $gt: 20 } })
Updatedb.students.updateOne({name:"John"}, {$set:{age:23}})
Deletedb.students.deleteOne({name:"Sara"})

7. MongoDB Query Language

Operators: $eq, $gt, $lt, $in, $and, $or, $exists

db.students.find({ age: { $gt: 20 }, city: "Delhi" })

8. Indexing

Indexes improve query performance.

db.students.createIndex({ name: 1 })

9. Aggregation

Used for data analysis and summarization.

db.students.aggregate([
  { $match: { age: { $gt: 20 } } },
  { $group: { _id: "$course", avgAge: { $avg: "$age" } } }
])

10. Sharding

Splits data across multiple servers for horizontal scaling.

  • Shard: Stores data
  • Config Server: Manages metadata
  • Mongos Router: Routes client queries

11. Join Operation

MongoDB uses $lookup for joins.

db.orders.aggregate([
  { $lookup: {
      from: "customers",
      localField: "cust_id",
      foreignField: "_id",
      as: "customer_info"
  }}
])

12. Pagination

Used to display results page by page.

db.students.find().skip(10).limit(5)

Summary

NoSQL databases like MongoDB are fast, flexible, and scalable. They store data in documents instead of tables, support complex queries, and handle massive data efficiently — making them perfect for modern web and cloud applications.
